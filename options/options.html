Custom Routes


FlowVPN can load all data for a website through a specific VPN server. This setting overrides the normal server selection in the popup.

For example: If you enter *google.com and select USA then all content for the website google.com will be loaded through a USA server where the web address for the main tab is *google.com

You can also force websites to load directly without using a VPN server.
